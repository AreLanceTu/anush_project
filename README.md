# Vivah Project

## Overview
Vivah is a comprehensive matchmaking and event management platform designed to facilitate partner selection, venue booking, and related services for users. The project includes user registration, profile management, chat functionality, booking integration, payment processing, and support features.

## Features
- User Registration and Login
- Partner Selection (Boys and Girls profiles)
- Venue Selection and Booking
- Payment Integration (Razorpay, Supabase, Firebase)
- Chat and Support System
- Shopping and Purchase History
- Upload and Profile Management
- Help and Support

## Folder Structure
- `anush_project/` - Main project directory
  - HTML pages for all features (Account, Cart, Chat, Matches, Registration, etc.)
  - JavaScript files for integrations and configuration
  - Firebase and Supabase configuration files
  - Payment integration scripts
  - Rules and indexes for database
  - Subfolders:
    - `30pagesofboys/` - Boys' profile pages and data
    - `30pagesofgirls/` - Girls' profile pages and data
    - `Boysimage/` - Boys' profile images
    - `GirlsImage/` - Girls' profile images
    - `images/` - General images
    - `script.js/` - JavaScript scripts
    - `shopping/` - Shopping-related files
    - `tools/` - Utility tools
    - `vivahvenue/` - Venue-related files

## Technologies Used
- HTML, CSS, JavaScript
- Firebase (Authentication, Firestore)
- Supabase
- Razorpay (Payment Gateway)

## How to Run
1. Clone or download the repository.
2. Open the project folder in VS Code or your preferred editor.
3. Configure Firebase and Supabase credentials in their respective config files.
4. Open `index.html` to start the application.
5. Use the navigation to access different features (registration, partner selection, venue booking, etc.).
